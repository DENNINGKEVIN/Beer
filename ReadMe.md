# BEER SUPERIOR
#### A app for beer research.
#### By **Denning Kevin Mulama**

## Description
This is an app which allows viewer to view different beer types.

## Setup/Installation Requirements
* To use this project, clone it.
* Open with android studio.
* Connect to phone or download android emulator.
* Run App.
